Multimedia

Multimedia types are sound, music, video, movies and animation. Video formats include: MPEG (.mpg or .mpeg), AVI (.avi), Quicktime(.mov), Flash (.swf or .flv) and MPEG-4 or MP4 (.mp4)

There are video attributes you can add: control, autoplay and width and height. Control adds the video control buttons like "play","pause" and "volume". Autoplay starts the video automatically. It is recommended to always include the width and height so the video doesn't flicker. Use the Google Chrome (or Safari) browser to best support the video format.

Use the HTML video tag to specify the video to be added. Use the source tag to define the media resource for the video or audio tag. If the 1st source doesn't play, the second will.

Cascading Style Sheets

We use CSS language to style the HTML pages. CSS describes how the elements should appear on the page. It allows you to add colors and font styles and a layout to the page.

CSS Declaration

CSS declaration consists of a selector the opening curly brace, property and a value. For example: h1 is a selector, it indicates the name of the tag you want to change the appearance of. It will affect all the tags with that name. The property is the change you make to the tag. For example the color, font type and font size are properties. Each property type is assigned a value. For example, red is a value of the property color. After adding multiple properties and values, close with a curly brace.

Ex: h1{color:red, font type:comic sans, size 16pt}

Add CSS to HTML

To add CSS, add a style tag within the HTML document inside of the HEAD tag, before the body of the code.

You can manipulate the properties for things like the color, font style(italicize), font weight (bold), font decoration(overline, line-through, underline or none. You can even underline and specify a color with underline blue). You can also customize the font size, the letter spacing, word spacing, the line height and the text-align. You can also transform the text using text-transform: capitalize, uppercase lowercase, or none.

The line height property sets the height of the lines in the tag withouth adjusting the size of the font. Curious to know what this looks like. Text align will align the text to the left, right or center depending on what you delegate.

You can also customize fonts by lengths and percentages. For example: em for the calculated size of a font(2em is twice the size of the current font size), px for pixels, pt for points, % for percentages, pc, cm for centimeters, mm for millimeters, in for inches

CSS Colors

CSS has access to over 16 million colors! Although there are only 17 preferred color names: They are aqua, black, blue, fuchsia, gray, green, lime, maroon, navy, olive, orange, purple, red, silver, teal, white, and yellow.

CSS colors can also be hexidecimal or decimal. The hexidecimal or hex code looks like this: #ff000. The decimal color code looks like this: rgb(255, 0,0)

The text color property is "color". The background color property is "background-color"

Using ID Attributes

ID attributes are used to specify a unique id for an HTML element. The id attribute points to the CSS style sheet. The id attribute value must be a unique name, since it adds style to one out of many of the same elements on a page. The name cannot be used more than once, and the name must be consistent on the CSS style sheet. Create a CSS style sheet by opening a new file called style.css. In the HTML file under the style tag, use a #theidname follower by curly braces.An id element can be used to style one pargraph to distingiush it from the other pargraphs.

Class Attributes

A class name can be used by multiple HTML elements (group).The style for a class name will be applied to all the elements that have that class name.
